# 智语助手 ZhiyuAI-Chat-Assistant

PC端AI聊天助手交互演示原型，课程结课作业项目

## 项目介绍

基于原生HTML+CSS+JavaScript开发，单文件离线运行，无需后端、无需大模型API，内置模拟对话逻辑。
完全匹配PRD产品需求与技术架构设计，实现核心演示功能：

1. 多会话新建、切换、管理
2. 快速模式 / 深度模式 双对话切换
3. 文字消息发送、AI模拟流式回复
4. 本地文件上传预览模拟

## 运行方式

1. 克隆代码到本地

```bash
git clone https://github.com/HHY-AFK/ZhiyuAI-Chat-Assistant.git
cd ZhiyuAI-Chat-Assistant
```
